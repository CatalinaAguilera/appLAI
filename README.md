# appLAI
Se modifica la ui en función para calcular indice de área foliar. 
También se recorta el área de interés aunque esté girado, pero se demora en exceso. 
Finalmente, se cambia la paleta de colores para que al calcular el LAI, se clasifique a las hojas con el color negro y fondo como blanco.
